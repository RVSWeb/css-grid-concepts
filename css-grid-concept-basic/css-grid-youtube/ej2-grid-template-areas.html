Podemos alinear cada 'Grid-Item' y diseñar el 'Grid-Layout' de forma manual utilizando la propiedad 'grid-template-areas' e identificando cada 'Grid-Item' mediante la propiedad 'grid-area' , una vez definido solamente tenemos que colocarlo en el 'Grid-Layout'
El 'Grid-Layout' tiene definido un ancho y un alto el cual si cada 'Grid-Item' ocupa mas espacio que el establecido cogera el espacio del 'Grid-Item' mas cercano a su posicion 

          grid-template-areas: 
          "item-2 item-2 item-3"
          "item-2 item-2 item-3"
          "item-2 item-2 item-3"
          "item-1 .      item-3"
