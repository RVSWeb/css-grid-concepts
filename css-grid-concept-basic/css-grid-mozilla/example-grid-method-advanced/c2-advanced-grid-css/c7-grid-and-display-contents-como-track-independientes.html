Ejemplo : Grid and display: contents

Hacer 'Elementos o Items Anidados' en 'Grid' actuen como si fueran parte de la 'Grid-Layout'

Es una forma de evitar problemas que las 'Sub-Cuadriculas o Sub-Grid' se resolverian una vez que se implementaran

Usar los 'display : content' de manera similar con 'flexbox' para permitir que los elementos anidados se conviertan en elementos flexibles 'Flex Item'
