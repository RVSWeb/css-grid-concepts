Grid and absolutely positioned elements

'Grid' interactua de forma absoluta posicionamiento de los elementos el cual puede ser util si deseas posicionar un 'Elemento o Item' dentro de un 'Grid' o una 'Grid-Area'

La especificacion define el comportamiento cuando un 'Grid Container' es un contenedor de bloques y un elemento 'Parent' tiene posicionamiento absoluto

A grid container as containing block

Para crear un 'Grid Container' con contenedores de bloques necesitas agregar la 'Propiedad' 'position : relative ' del contenedor 'Parent' Igual que se haría con un bloque de 'Container' para cualquier otro elemento en 'position: absolute'

Hecho esto , si tu le das a los 'elementos hijos' o 'Grid Items ' la propiedad 'position: absolute' , ello hara que el 'Grid Items ' lo tomes como un contenedor de bloques el 'Grid Container' o si el 'Grid Item' tambien tiene un 'Grid Position '
El area de la 'Grid' sera donde se coloque.

En el ejemplo tenemos un 'Grid Container' con 4 'Elementos hijos'
Los 3 elementos o '3 Items' estan absolutamente posicionados y colocados sobre 'Grid Container' usando la linea de 'line-based' para colocarlo
El 'Grid Container' tiene posicionamiento 'position: relative' asi se convertira en el posicionamiento del contexto de este 'Elemento' o 'Item'
