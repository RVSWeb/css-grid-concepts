Fr unit and Flex Basis

Fr Unit

Asigna una porción de espacio disponible dentro de un 'Grid Container' para nuestra 'Grid Tracks o Elementos en forma de franjas'

Cuando 'Fr Unit' la combinamos con la funcion minmax() da un comportamiento similar a las propiedades 'flex' dentro de 'FlexBox' mientras permita la creación de un diseño de 2 dimensiones

Flex Layout

Si arrastramos la ventana desde lo amplio a lo mas pequeño el 'Flex-Box' se ajusta al numero de 'Elementos' dentro de cada fila acorde al espacio 'Flex-box ideal para elementos interiores' porque se ajusta al espacio interior del contenedor principal.

Si tenemos un monton de espacio para 5 'Elementos o Items' podemos fijar todos en una sola fila. Si tenemos un contenedor muy estrecho solo tendremos espacio para un 'Elemento o Item'

En comparacion la version 'Grid-Layout' siempre tiene 3 columnas de 'Track' que ellos mismos creceran o encogeran pero siempre habra 3 elementos porque son los definidos por nuestra cuadricula

Auto-filling grid tracks

A flexible number of tracks

Cuando los 'Elementos' o 'Item' superan el largo de la envoltura 'wrapping'

Podemos alcanzar todo el ancho de los 'Elementos' igual que con las propiedades Flex-box usando 'Grid-Layout' combinando 'auto-fill' y la funcion 'minmax()'

Vamos a crear el auto llenado de 'Track' usando funcion 'minmax()' el cual los 'Track' tendran un minimo de '200px' asi que establecemos el maximo en '1fr'.

Una vez que el navegador ha resuelto cuántas veces caben 200 píxeles en el contenedor 'Container' para llenarlo tambien considerando los huecos entre cada 'Grid-Gaps' , el navegador intentara el maximo '1fr' en cada instruccion para compartir el espacio entre los restantes elementos

Ahora tenemos la capacidad de crear una cuadrícula con un número flexible de pistas flexibles, pero al mismo tiempo ver los elementos dispuestos en la cuadrícula alineados por filas y columnas.

display: grid;
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
grid-auto-rows: 100px;
display: grid;
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
grid-auto-rows: 100px;
display: grid;
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
grid-auto-rows: 100px;
