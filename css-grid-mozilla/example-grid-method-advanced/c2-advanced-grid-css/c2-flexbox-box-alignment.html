Box alignment

Flexbox - Layout

Facilita centrar un 'Box' en la pagina web Los Flex 'Item' o 'Elementos' se puede estirar sobre el contenedor del Flexbox Layout. Podemos ajustar el alto 'height' de las columnas automaticamente

Cuando usar Flexbox - Layout

Cuando tenemos un contenedor 'container' con 3 elementos 'items' dentro. El wrapper 'min-height' esta establecida y define el alto de los 'Flex Container'
Establecemos 'align-items' sobre 'Flex Container' a 'flex-end' asi que los elementos se alinean hasta el final del 'flex container'
Tenemos que establecer los 'align-self' sobre 'box1' asi que sobreescribiremos por defecto el estiramiento de la altura del contenedor y sobre la 'box2' asi que ellos se alinean al comienzo de las 'Flex Container'
align-self: stretch;
align-self: flex-start;
align-self: center;
align-items: flex-end;
