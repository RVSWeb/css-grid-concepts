Diferencia entre Flexbox Layout & Grid Layout

Usar Flexbox Layout
Para crear el diseño de filas o columnas unicamente.

Usar Grid Layout
Para crear el diseño por fila y columna conjuntamente.

Utilizar Flexbox en contenidos cuando tienes un conjunto
de elementos y deseas espaciarlos
uniformemente en un contenedor
Dejas que el tamaño del contenido decida
cuánto espacio individual ocupa cada elemento.
Si los elementos se ajustan a una nueva línea
calcularán su espacio según su tamaño y el
espacio disponible en esa línea.
