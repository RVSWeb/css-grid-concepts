Grid and display: contents

Utilizar Grid-Layout con 'display : contents'
La propiedad 'display : contents' en si misma no genera una 'Cajas o Boxes' pero sus descendientes hijos y pseudo-elementos generan 'Cajas o Boxes' normales.

El elemento sí mismo no genera ninguna 'Cajas o Boxes, pero sus 'Elementos o Items Hijos' descendientes y 'pseudoelementos' todavía generan 'Cajas o Boxes como normales. Para los objetivos de generación de 'Cajas o Boxes y disposición, el elemento debe ser tratado como si había sido sustituido por sus 'Elementos o Items Hijos' y 'pseudoelementos' en el árbol de documento

Si configuras un 'Elemento' o 'Item' como
'display : contents'

La 'Caja o Boxes' que se crearia desapareceria y la 'Caja o Boxes' de los elementos hijos apareceria como si ellos hubieran subido de nivel
Significa que el elemento hijo de los 'Grid Items' se pueden convertir en 'Grid Items'

Tenemos un 'Grid' y el 'Primer elemento' sobre el 'Grid' se ajusta para abarcar el total de los 'Track' en tres columnas.

Los cuales contienen 3 'Items o Elementos' anidados. Como esos 'Items o Elementos' no son directamente hijos 'no se convierten en parte de los Grid-Layout' y se muestra como 'Bloques o Boxes ' de 'Layout'
