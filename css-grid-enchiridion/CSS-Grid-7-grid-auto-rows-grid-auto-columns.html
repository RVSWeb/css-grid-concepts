Grid with grid-auto-rows

grid-auto-rows requires grid-auto-flow: row (default) to create rows for auto-placed items.

grid-auto-rows: 50px 100px 200px

Automatically creates rows in the sizes 50px, 100px, 200px repeatedly until all items are in the implicit grid.

Grid with grid-auto-columns

grid-auto-columns requires you to set grid-auto-flow: column to create columns for auto-placed items.

grid-auto-columns: 50px 100px 200px

Automatically creates columns in the sizes 50px, 100px, 200px repeatedly until all items are in the implicit grid.
